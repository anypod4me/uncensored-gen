# ⚡ FLUX UNLIMITED 4K

A high-performance, unrestricted AI image generation interface powered by the **FLUX.1** model via Pollinations AI.

## 🚀 Features
- **Unfiltered Creative Freedom**: No built-in safety filters for raw artistic output.
- **4K Resolution**: Generates high-fidelity images at 2048x1152 resolution.
- **Ultra-Fast**: Optimized for rapid generation with real-time loading states.
- **Zero Cost**: Completely free to use with no API keys required.
- **Modern UI**: Dark-mode aesthetic built with Next.js, Tailwind CSS, and Lucide Icons.

## 🛠️ Built With
- [Next.js](https://nextjs.org) (App Router)
- [Tailwind CSS](https://tailwindcss.com)
- [Lucide React](https://lucide.dev) (Icons)
- [Pollinations AI](https://pollinations.ai) (API)

## 📦 Deployment
This project is configured for **GitHub Pages** using a static export.

1. Clone the repo: `git clone https://github.com`
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`
4. Build for production: `npm run build`
